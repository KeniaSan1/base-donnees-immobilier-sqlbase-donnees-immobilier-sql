# Base de données immobilière et analyse du marché immobilier français

## Contexte

Ce projet a été réalisé dans le cadre de la formation Data Analyst d'OpenClassrooms.

L'objectif était de concevoir une base de données relationnelle à partir de données immobilières françaises et de répondre à des problématiques métier à l'aide de requêtes SQL.

## Objectifs

* Concevoir un modèle de données cohérent et normalisé.
* Créer un dictionnaire de données.
* Élaborer un modèle UML et un schéma relationnel.
* Structurer des données issues de plusieurs sources.
* Répondre à des questions métier à l'aide de requêtes SQL.

## Outils utilisés

* SQL
* Excel
* UML
* Modélisation de données

## Compétences développées

* Data Modeling
* Conception de bases de données relationnelles
* Normalisation des données
* SQL
* Data Analysis
* Documentation technique

## Résultats

* Création d'une base de données immobilière structurée.
* Élaboration d'un dictionnaire de données et d'un modèle UML.
* Développement de requêtes SQL pour analyser les transactions immobilières.
* Analyse des prix de vente et des tendances du marché immobilier français.
